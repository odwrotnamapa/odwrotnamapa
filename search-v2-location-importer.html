Importer lokalizacji OSM

Importer działa jako skrypt Node.js, ponieważ przeglądarka nie może bezpiecznie zapisywać plików projektu.

Test na danych przykładowych

npm run location:import:sample

Import własnego Overpass JSON

node tools/update-location-db.cjs --input ścieżka/do/pliku.json

Pobranie z Overpass API

npm run location:download

Po imporcie skrypt automatycznie uruchamia walidację i buduje pliki w katalogu search-v2/location/compiled/.
